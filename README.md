COMO EXECUTAR O SISTEMA: 

1-Ter o XAMPP baixado no computador;
2-Ligar o Apache;
3-Encontrar a pasta do Xampp nos arquivos e abri-la;
4-Encontrar a pasta "htdocs" dentro da pasta xampp;
5-Dentro da pasta "htdocs", cole a pasta do sistema edufluxo;
6-Ir no google e escrever "http://localhost/Sistema-EduFluxo;
7-Clicar em "PROJETO".
